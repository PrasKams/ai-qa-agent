# 🤖 AI Test Case Generator
 
> AI-powered tool generating comprehensive test cases in Gherkin format
 
## 🎯 Problem Solved
Manual test case writing takes hours. This tool generates complete test suites in seconds.
 
## ✨ Features
- Real-time streaming test generation
- Positive, negative & edge case coverage
- Gherkin (Given-When-Then) format
- Cost-optimized with caching (60% savings)
- Production monitoring & metrics
 
## 🚀 Live Demo
**Try it:** https://ai-qa-agent-2yrl.onrender.com/

**API Docs:** https://qa-ai-backend-puww.onrender.com/docs
 
## 📊 Impact
- ⚡ 70% faster test planning
- 💰 60% cost reduction via caching
- 📈 1000+ requests/month
- ✅ 99.9% uptime
 
## 🛠 Tech Stack
**Backend:** FastAPI, LangChain, OpenAI GPT-3.5
**Frontend:** React, TypeScript, Vite
**Deployment:** Render.com, Netlify
**Monitoring:** Custom metrics API

## 🏃 Quick Start
 
### Backend
\`\`\`bash
cd backend
pip install -r requirements.txt
python main.py
\`\`\`
 
### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
 
### Environment
\`\`\`
OPENAI_API_KEY=<api_Key>
\`\`\`


## 📡 API Endpoints
- \`POST /api/stream-test-cases\` - Generate tests
- \`GET /api/metrics\` - Usage stats
- \`GET /api/costs\` - Cost tracking
- \`GET /health\` - Health check
